Traceability Supply Chain

This application demonstrates the basic scenario of traceability supply chain

The producer will produce the shipment and hand it over to shipper for shipping. The shipper will place sensor on the shipment and will start its journey.

The sensor will periodically update the current location of the shipment to the blockchain.

The receiver of receiving the shipment can check the journey of the shipment and mark shipment as received.

The control authorities can verify the final shipment data and decide to punish producer/shipper and issue refund to the receiver

Please install metamask and select rinkeby network

Flow of the application

  1. Producer adds Shipment
  2. Shipper adds sensor
  3. Shipper starts the shipment
  4. Sensor updates the location of shipment regularly
  5. Receiver tracks the location
  6. Receiver Receives the shipment
